## Quick start

### Setup
```console
root@PC:~$ git clone https://github.com/ppostnov/slb_task.git
root@PC:~$ cd slb_task
root@PC:~$ docker-compose up airflow-init
root@PC:~$ docker-compose up
```
### Usage
#### Swagger UI
```console
http://localhost:8000/docs
```
#### Airflow UI
```console
http://localhost:8080
```
